# pattern-math-dsa
A collection of pattern printing and basic mathematical problems implemented in C++ for beginners to strengthen logic-building and problem-solving skills.
# 🔢 Pattern + Math DSA (C++)

## 📖 Description
This repository contains pattern printing and mathematical problems in C++ for beginners.

## 🚀 Features
- Prime Number Check
- GCD using Euclidean Algorithm
- Armstrong Number
- Palindrome Number
- Star Patterns
- Number Patterns

## 🛠️ Technologies Used
- C++

## 📂 How to Run
1. Clone the repository
2. Open in any C++ IDE
3. Compile:
   g++ main.cpp -o pattern
4. Run:
   ./pattern
